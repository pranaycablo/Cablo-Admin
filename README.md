# Cablo: Global AI-Driven Mobility Platform

Cablo is a next-generation SaaS platform for multi-mobility services, powered by a swarm of AI agents.

## Project Structure

- `backend/`: Node.js + Express API (The AI Brain)
- `frontend/`: React Native (Expo) mobile application for Users, Captains, and Admin.
- `docs/`: Architecture and design documentation.

## AI Agents (The Brain)

1. **AI CEO**: Strategy & Orchestration
2. **AI COO**: Operations & Allocation
3. **AI CFO**: Finance & Pricing
4. **AI CTO**: System & Security
5. **AI Manager**: Growth & Engagement

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm start`

### Frontend
1. `cd frontend`
2. `npx expo start`

## Tech Stack
- **Backend**: Node.js, Express, Socket.io, MongoDB, BullMQ
- **Frontend**: React Native, Expo, Mapbox
- **AI**: Agentic workflow with human-in-the-loop learning.
